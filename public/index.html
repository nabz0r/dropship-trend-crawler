API REST

Le système expose les endpoints suivants pour l'intégration avec d'autres services :

  • GET /api/products - Liste des produits
  • GET /api/trends - Tendances actuelles
  • GET /api/trends/history - Historique des tendances
  • GET /api/trends/categories - Tendances par catégorie
  • GET /api/settings - Paramètres du système

Consultez la documentation dans le dossier docs/ pour plus d'informations.
